    Метод give

    Блокчейн: Все
    Стоимость: Бесплатно если адрес был выдан ранее или 1 единица + 1 единица если адрес виртуальный

    Выдает адрес для входящей оплаты. Возвращает адрес, его публичный ключ и (если разрешено настройками) его приватный ключ. Обычно такие адреса являются временными или транзитными
    Если передать адрес в параметре address или в uniqID (или label) значение от адреса, который находится в архиве, то API восстановит и вернет этот адрес в работу
    Параметры:
    *token - дополнительно отображать баланс этого токена у адреса
    *from - с этого адреса брать комиссию на пересылку токенов [основной адрес]
    *to - на этот адрес пересылать монеты [из настроек или основной адрес]
    *statusURL - URL обработчика IPN [из настроек]
    если указать ‘-’, то для этого адреса IPN не будет отправляться
    *label - метка, передается в IPN
    *uniqID - уникальный ID запроса, которому соответствует выдаваемый адрес [если не указан, то используется значение label].
    если повторить uniqID, то API вернет выданный ранее адрес,
    но остальные параметры адреса обновятся
    *waitPeriod - время до архивирования в минутах (макс. период ожидания поступления средств)
    *period - время до архивирования в минутах (макс. период ожидания следующего поступления средств)
    *reusable - использовать в пуле адресов (см п.7) [0 - нет]
    *forceGenerate - принудительно сгенерировать новый [0 - нет]
    *address - создать “виртуальный” адрес (см п.9) или вернуть из архива
    *privateKey - создать адрес из такого приватного ключа
    *publicKey - создать виртуальный адрес из такого публичного ключа
    *isPrimary - созда��ь основной адрес [0 - нет]
    *qr - возвращать картинку QR-кода [0 - нет]
    Пример вызова:
    https://new.cryptocurrencyapi.net/api/trx/.give?key=ac46bb60ed-a21e9a46ca-227f212121-b490a95afc&label=u1&period=30
    Пример ответа:
    {"result":{"address":"TKavpKP2VJbfV4AGyi3MrhT6FJAP5eJ2RR","publicKey":"03b7a403f5ffce0292b8f17328b7f0dd0ca52ac5549292bc6fe1fe580ef40d183f","privateKey":"***"}}
