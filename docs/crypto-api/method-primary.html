    Метод primary

    Блокчейн: Все
    Стоимость: Бесплатно

    Переключает основной (primary) адрес
    Параметры:
    address - адрес, который должен стать основным
    Пример вызова:
    https://new.cryptocurrencyapi.net/api/trx/.primary?key=ac46bb60ed-a21e9a46ca-227f212121-b490a95afc
    Пример ответа:
    {"result":true}
